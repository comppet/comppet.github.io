Componentes do PrimeFaces(){
Dicas

Veja a seguir alguns dos componentes do PrimeFaces que merecem destaque!

Mock OS X

A coletânia de componentes do Mock OS X é um conjunto de componentes que imitam a interface do sistema operacional da Apple, Macintosh. Acesse http://www.primefaces.org/showcase/ui/macosx.jsf para testar a interface.

Calendário

O calendário é uma interface bastante comum hoje em dia e o PrimeFaces não deixa de implementá-la. Além do calendário comum, a versão 3.0 do PrimeFaces inclui um seletor de horário no componente.

Drag and drop

Um dos recursos mais comuns em aplicações desktop e raras em aplicações web é o "Drag and Drop", ou "Arrastar e Soltar" em português. Com o Primefaces é possível implementá-lo com poucas linhas de código, permitindo que o usuário arraste os painéis do website deixando a interface ao seu gosto.

Data Table

Como o próprio nome diz, esse componente apresenta uma tabela com um conjunto de dados. Essa tabela pode facilmente incluir paginação, busca e até edição em célula.

Google Maps

Com o PrimeFaces é fácil integrar sua aplicação com essa potente ferramenta do Google.

Editor

O editor é uma caixa de texto especial que permite ao usuário formatar um texto e até inserir imagens sem utilizar se quer uma linha de HTML.

Growl

Pra que alertar o usuário sobre o sucesso ou erro de uma operação com simples mensagens de texto quando basta digitar "<p:growl>" para ter caixas de texto flutuando de forma elegante em sua aplicação?

Upload

Meio chato abrir uma janela para que o usuário selecione um arquivo para fazer upload não? Porque não fazer igual no sistema operacional onde apenas arrastamos e soltamos os arquivos no local para onde eles devem ser enviados?

Para verificar todos os componentes do PrimeFaces acesse www.primefaces.org/showcase/.

Além dos componentes próprios para serem exibidos em PCs, o PrimeFaces conta também com uma suíte de componentes para dispositivos móveis com touch screen, facilitando o desenvolvimento de aplicações web para tablets e smartphones. Acesse www.primefaces.org/showcase/touch/ para visualizar os componentes (a melhor maneira de testar os componentes é através de um dispositivo móvel).
