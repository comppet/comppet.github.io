Software Livre

Outro projeto que surgira a partir do Mozilla Suíte, o Thunderbird é o cliente de e-mail da Mozilla, sendo alternativa a softwares consagrados do gênero, como é o caso do Microsoft Outlook. Com este programa, o usuário é capaz de importar seus e-mails, seja ele de qual servidor de e-mail for. A aposta do Thunderbird para competir com seu rival da Microsoft, além de sair de graça para o usuário, é a gama de funcionalidades adicionais que não deixam de ser interessantes ao usuário, como é o caso do filtro de spams inteligente, leitor de notícias RSS e ainda catálogo de endereços com suporte à utilização de imagens, além disso o software da Mozilla é extremamente simples de se utilizar. Um dos poucos contras desse software é o seu alto consumo de memória. O Thunderbird se encontra em sua versão 5. Para fazer o download dessa ótima ferramenta, acesse: http://www.mozilla.org/pt-BR/thunderbird/
