Tor Messenger: O Mensageiro Anônimo(){
Notícias

Por: Felipe Brandão


O projeto TOR é um dos mais incríveis e populares quando se tem como requisito a privacidade e anonimato na Web (especialmente Deep Web). De acordo com o site ArsTechnica, os responsáveis pelo projeto estão agora a desenvolver o seu próprio serviço de mensagens instantâneas, que permitirá a comunicação em tempo real, preservando sempre o anonimato dos utilizadores (recorrendo para isso à rede TOR). Este serviço/cliente vai ser chamado Tor Instant Messaging.

O "Tor Messenger" utilizaria a rede anônima para transmitir a mensagem de um usuário para outro. O software poderia ser disponibilizado sozinho, de forma independente ou incluído no Tor Browser Bundle, que permite que usuários utilizem um navegador já configurado para navegar de forma anônima.

Ele será desenvolvido sobre o Instabird, um mensageiro instantâneo de código aberto. O desenvolvimento incluirá criptografia OTR (Off-the-Record Messaging) e as mensagens passarão pela rede Tor. A expectativa era de que ele estivesse pronto ainda em Março, mas o projeto não inclui um prazo final.

O site dos desenvolvedores, onde o projeto foi identificado pelo Daily Dot, diz que auditores independentes deverão dar seu aval para a plataforma. O aplicativo também deve ser traduzido para diversas línguas, entre elas as árabes, em que há uma censura maior ao ativismo político. Assim, eles poderiam se comunicar de forma mais segura.

Os responsáveis pelo projeto TOR tem trabalhado no Tor Instant Messaging, desde Julho do ano passado, altura em que começaram vir a público vários escândalos de espionagem levados a cabo pela agência de segurança Norte Americana (NSA). Sabe-se também que este serviço de mensagens está sendo patrocinado por um anônimo, que apenas se identifica como “O Patrocinador”.
