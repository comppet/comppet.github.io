Desenvolva IA com projetos de código-aberto!

Existem diferentes formas de estudar e desenvolver Inteligência Artificial(IA). Na internet, pode-se achar vários projetos de desenvolvimento de IA. Uma ba dica para estudar IA, é pegar códigos de programas já feitos enterder o funcionamento e a lógica do mesmo. Se o interesse for além de apenas estudos, pode-se ajudar a desenvolver e aprimorar projetos de IA disponibilizados na internet. Abaixo estão alguns desses projetos com códigos-fonte de seus programas.

O OpenCog é um projeto ambicioso de software open-source com o objetivo de criar um framework para Inteligência Artificial geral. O OpenCog tem como obejtivo fornecer um framework para desenvolvimento de inteligência artificial para desenvolvedores e cientistas. O código do OpenCog pode ser encontrado no seguinte link.

O OpenROV é um open-source de baixo custo. Basicamente constitui- se de um robô subaquático para exploração do oceano. Como sendo software livre, pode-se utilizar o OpenROV para estudo de inteligência artificial. Para mais detalhes do projeto, acesse o seguinte link .

O projeto FreeHAL tem como objetivo desenvolver uma inteligência artificial de código aberto capaz de imitar a conversação humana. O código-fonte do projeto pode ser encontrado no GitHub, porém no site oficial do projeto você encontrar novas ideias para o projeto e ajudar a desenvolver e aprimorar cada vez mais o programa. O site oficial pode ser encontrado no seguinte link.
