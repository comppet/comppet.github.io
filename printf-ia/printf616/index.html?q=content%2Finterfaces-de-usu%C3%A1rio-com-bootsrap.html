Por: Yassin Nurmahomed


O Bootstrap é um conjunto de ferramentas open-source desenvolvido para facilitar a criação de webapps e websites. É constituído de elementos de interface de usuário, layouts e ferramentas para desenvolvimento em javascript compactados em um arquivo zip de apenas 285.3kb (a versão 2.0.2). A ferramenta surgiu devido a necessidade de diminuir as inconsistências existentes entre projetos individuais dos engenheiros do Twitter, já que no inicio cada um desenvolvia usando ferramentas com as quais já estavam mais familiarizados.

Durante a primeira Hackweek do Twitter seu desenvolvimento foi acelerado e pelo final da semana tinha se alcançado uma versão estável que podia ser utilizada pelos engenheiros da companhia. A primeira versão do Bootstrap foi lançada em Agosto de 2011 sob a licença Apache 2.0 que permite que qualquer pessoa possa modificá-lo e customizá-lo do modo que entender, para depois utilizá-lo em em projetos de carácter grátis ou lucrativo desde que atribua mérito aos desenvolvedores do Twitter.


Construído com Less


O Boostrap é no seu núcleo nada mais que CSS puro construído através do Less, um pré-processador que oferece mais flexibilidade que o CSS convencional, permitindo o uso de variáveis, mixins (propriedades de classes dentro de outras classes), operações e funções. O Less pode ser compilado client-side via Javascript ou server-side via Node.js ou Rhino. Depois de compilado o Bootstrap contém apenas CSS sem imagens e sem Flash. Existe também a versão compilada que pode ser baixada diretamente do site oficial, mas para customização é recomendável modificar diretamente o Less ou baixar uma versão personalizada, com apenas aos componentes que se pretende usar, pelo site oficial.
