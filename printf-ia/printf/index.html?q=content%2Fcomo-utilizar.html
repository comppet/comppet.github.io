Como utilizar?(){
Dicas

Utilizaremos o NetBeans no tutorial abaixo.

  1. Baixe a biblioteca PrimeFaces em www.primefaces.org/downloads.html.

  2. Crie um novo projeto Java Web e lembre-se de no último passo adicionar o framework JSF.

  3. Inclua, nas bibliotecas do projeto, o arquivo correspondente ao jar do PrimeFaces.

  4. Crie uma classe java que fará o controle do componente (ManagedBean).

  5. Na página em que deseja utilizar os componentes do PrimeFaces adicione o seguinte endereço na tag <html>: xmlns:p="http://primefaces.prime.com.tr/ui.

  6. Pronto! Para utilizar um componente do PrimeFaces basta utilizar a tag <p:componente> em conjunto com o ManagedBean. Veja no próximo passo como fazer um simples campo para a entrada de texto.

  7. Utilize o componente e execute o projeto.

Este é um exemplo simples, mas todos os outros componentes do PrimeFaces são de fácil utilização e proporcionam ao seu sistema uma interface incrível. Vale a pena conferir todos os componentes desta biblioteca! Para visualizar os componentes e aprender como eles funcionam acessem http://www.primefaces.org/showcase/.
