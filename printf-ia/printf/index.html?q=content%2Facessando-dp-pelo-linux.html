Acessando a DW pelo Linux(){
Dicas

Por: Raphael Cardoso Fernandes

Como foi falado anteriormente, neste tópico iremos te ensinar como acessar a DW, vamos usar o Slackware, que é uma distribuição GNU/Linux, mas não há segredo para outras distribuições, os passos são basicamente os mesmos.

Aí vão os passos:
1- Acesse o site do Tor: torproject.org e clique em Download Tor. Ou simplesmente clique aqui: Download Tor.
2- Agora, escolha entre 32bits ou 64bits, a depender do computador e da distribuição que você está usando. Caso você não saiba, faça o seguinte teste:
2.1 - Abra o console e digite: uname -m. Se estiver usando Slackware, irá aparecer:
bash-4.2$ uname -m
i686
2.2 - Perceba que meu sistema é 32bits, caso o seu seja 64bits, irá aparecer:
bash-4.2$ uname -m
x86_64
3 - Após o Download você receberá um arquivo no formato .tar.xz salvo na sua pasta de Downloads padrão.
4 - Ainda no console, acesse sua pasta de Downloads e descompacte o arquivo usando:
4.1 - tar -xvJf tor-browser-linuxVER-3.6.1_LANG.tar.xz, onde VER você deve substituir pela versão do seu sistema e LANG que você deve substituir pela linguagem do arquivo. Ou simplesmente, digite tar -xvJf, digite um pedaço do nome do arquivo e pressione Tab para o auto completar.
4.2 - Entre na pasta descompactada, usando: cd tor-browser_LANG, onde LANG é a linguagem do arquivo. Você pode repetir os passos acima no que se diz respeito ao Tab.
4.3 - Ao acessar a pasta, ainda no console, digite ./start-tor-browser
5 - Uma janela irá aparecer, você deve escolher Connect para acessar diretamente ou escolha Configure caso sua rede possua algum tipo de Firewall, proxy, etc.
6 - Após configurar ou conectar diretamente, um novo navegador será iniciado e então bingo! Você estará conectado na Deep Web.
