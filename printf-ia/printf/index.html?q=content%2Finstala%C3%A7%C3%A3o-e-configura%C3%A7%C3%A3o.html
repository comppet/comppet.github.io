Instalando o Samba no Ubuntu


Para instalar o Samba no Ubuntu, basta abrir o terminal e copiar este comando:

$ sudo apt-get install samba samba-common python-glade2 system-config-samba smbclient

Após o fim da instalação, será necessário criar um usuário no Samba, com o comando:

$ sudo smbpasswd -p nome_do_usuario_ubuntu

Onde "nome_do_usuario_ubuntu" deve ser o nome do usuário no Ubuntu.
Pronto! Agora já é possível compartilhar dados e impressora entre Ubuntu e Windows. O Samba pode atuar em uma rede de computadores de duas formas:

  • Como servidor: é uma máquina responsável por fornecer serviços a outras máquinas. Caso queira compartilhar arquivos no Ubuntu, e acessá-los em um outro computador, pelo Windows, veja como configurar o Samba como servidor de dados.
  • Como cliente: este, por sua vez, consome os serviços fornecidos pelo servidor. Para aprender como ter acesso, no Ubuntu, a arquivos e impressora compartilhados pelo Windows, veja a seção da matéria sobre como usar o Samba como cliente de dados e de impressora.
