Ubuntu 10.10: O novo Maverick Meerkat(){
Notícias

Por: Daniel Alves

Tela de instalação apresenta alguns recursos da versão.

Está disponível, desde o dia 10 de outubro, a versão 10.10 do sistema operacional Ubuntu, batizada de Maverick Meerkat (algo como o suricate rebelde em português). O lançamento, que, normalmente, acontece ao final dos meses de abril e outubro, desta vez foi antecipado. O dia escolhido se deve não apenas por coincidir com o número da versão, mas também celebra uma data que se repete a cada 100 anos (10/10/10) e que pode ser lido como a representação binária do número 42, que, na série "O Guia do Mochileiro das Galáxias", é a resposta para a pergunta fundamental sobre a vida, o universo e tudo mais.

Curiosidades à parte, a versão chegou repleta de novidades para facilitar a vida do usuário. A primeira delas é vista logo no primeiro contato com o S.O. O menu de instalação desta versão está mais amigável e, enquanto se acompanha a barrinha de progresso, são exibidas imagens e textos para que o usuário se familiarize com os recursos disponíveis, bem ao estilo Windows mesmo.

Controle integrado do Rhythmbox, o detalhe é na nova fonte
do sistema.

Ao logar no sistema é possível notar algumas leves mudanças no tema, uma alteração sutil, mas que faz diferença, é a nova fonte, que tem um visual mais suave do que a utilizada até então. Outra mudança legal foi a integração do player Rhythmbox com as opções de áudio do sistema, tornando mais fácil trocar de faixas ou visualizar informações sobre a música que está tocando.

A versão conta também com uma edição para netbooks, que traz uma interface de usuário Unity, adequado para pequenas telas. As duas edições trazem forte integração com a Ubuntu One, projeto da Canonical que disponibiliza "nuvens" pessoais para armazenamento de arquivos. Desta forma é possível sincronizar dados com outros computadores ou celulares, bem útil.

Para saber mais sobre a nova versão do Ubuntu acesse:
http://www.ubuntu-br.org

Confira também os serviços Ubuntu One:
https://one.ubuntu.com (em inglês)
