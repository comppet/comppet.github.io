Brasil terá aeronaves com rede Wi-Fi(){
Notícias

Por: Andrêssa Finzi


Já imaginou acessar sites, verificar seu e-mail, ver notícias, tudo isso durante uma viajem de avião? Essa realidade está cada vez mais próxima de nós brasileiros através do serviço OnAir oferecido pela empresa aérea TAM que permite usar celular a bordo. Já são quatro aeronaves com esse serviço com velocidade do acesso via celular de EDGE/GPRS e a tendência é esse número crescer mais ainda. A TAM promete 31 aviões com o serviço instalado para rotas nacionais até 2012. Em outros países este serviço é mais frequente e funciona basicamente através de um retransmissor que se conecta a redes por satélite.

A partir do segundo semestre de 2012 os vôos internacionais, terão quatro aeronaves com sinal de celular e Wi-Fi inicialmente, mas a previsão é conter mais oito aviões para ser entregues até 2014. O motivo de se expandir para vôos internacionais é explicado pelo Jósé Zaidan Maluf, da TAM: "Nossos passageiros querem estar conectados durante os vôos. Percebemos isso com a popularidade dos serviços OnAir em nossas aeronaves de rotas domésticas. Não há dúvida de que a conectividade a bordo se tornará onipresente e estamos determinados a liderar esse movimento."

Essa opção faz muita diferença para quem viaja a trabalho e precisa adiantar algo durante a viajem, ou simplesmente para quem quer se distrair durante uma viajem longa. O único problema é o preço, mesmo nos vôos domésticos as tarifas são de roaming internacional e são cobradas diretamente pela operadora com isso fica muito caro usar o celular. O valor para usar o Wi-Fi ainda não foi definido.
