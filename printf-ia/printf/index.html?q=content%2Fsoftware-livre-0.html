Software Livre

O que é um Software livre?

Software livre, segundo a definição criada pela Free Software Foundation, é qualquer programa de computador que pode ser usado, copiado, estudado e redistribuído sem restrições. O conceito de livre se opõe ao conceito de software restrito (software proprietário), mas não ao software que é vendido almejando lucro (software comercial). A maneira usual de distribuição de software livre é anexar a este uma licença de software livre, e tornar o código fonte do programa disponível.

Nessa revista traremos três matérias interessantes. A primeira delas será sobre uma das empresas de Software Livre mais respeitadas no mundo e mais bem sucedidas. A segunda falará sobre um programa que ajuda as pessoas a compreenderem o tráfego de informações na rede. E por último, um programa que faz qualquer programa do Windows funcionar em qualquer outro sistema operacional.
