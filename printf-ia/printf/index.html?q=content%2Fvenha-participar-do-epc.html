Venha participar do EPC!(){
Notícias

Por: Eder Paraiso C. Flor


EPC é o grupo de “Estudo de Problemas Computacionais” e tem por objetivo discutir soluções para diversos desafios de computação. Os problemas são retirados das maratonas de computação e são uma oportunidade não só para conhecer melhor o funcionamento de uma competição dessas, como também para desenvolver o raciocínio para resolver problemas.
Com a coordenação da profa. Dra. Márcia Aparecida Fernandes, os encontros do grupo ocorrem semanalmente na terça-feira às 19h30 com término às 21h. Você que é aluno de computação ou interessado em desafios que envolvem o raciocínio lógico venha fazer parte do grupo! Entre em contato conosco para saber o local de cada encontro. Encare este desafio!
