Software Livre

O Firebug é uma extensão para o Mozilla Firefox e é uma ferramenta indispensável para qualquer desenvolvedor web, uma vez que este software disponibiliza diversas funcionalidades para a depuração de um site e monitoramento de códigos html, javascript e css. Em qualquer página web que você esteja é possível inspecionar o código do mesmo. De simples utilização, basta, por exemplo, selecionar a opção inspecionar do Firebug e depois "passear" com o cursor pela página atual e teremos a porção de código html de cada div em que você visite com seu mouse, além disso, uma vez que você tenha identificado alguma div ou qualquer outro componente html, se o mesmo fizer parte de uma classe de estilo definida em algum arquivo css do site é possível ver o código css da classe em questão e fazer alterações, claro que provisórias (não serão replicadas no arquivo original), para ver como ficaria um fundo com uma cor diferente, um menu posicionado em outro lugar, uma fonte maior no texto do site, entre outros testes. O Firebug ainda possui diversas funcionalidades, como um console onde erros da página podem ser encontrados, etc. Faça o download dessa ótima ferramenta em http://getfirebug.com/
