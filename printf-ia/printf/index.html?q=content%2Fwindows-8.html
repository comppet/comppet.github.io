Windows 8(){
Capa

Por: Yassin Nurmahomed


Em Junho a Microsoft fez uma demonstração da nova geração do seu sistema operacional de codinome Windows 8, sendo este não necessariamente o nome de sua versão final. Considerado como uma reimaginação do Windows, o novo sistema apresenta inovações em todos os níveis de sua arquitetura desde o kernel a interface gráfica.

Como foi anunciado no inicio do ano, o sistema operacional irá suportar arquiteturas SoC (System on a Chip) presentes na maioria dos dispositivos móveis atuais como o chip Snapdragon da Qualquomm, expandindo a sua abrangência para além dos Desktops.

A nova interface gráfica é caraterizada por ser voltada para dispositivos touch, o novo start menu apresenta em vez de ícones, telhas, que sendo maiores dispõem de mais espaço permitindo que as aplicações apresentem informações relevantes sem que o usuário precise executá-las. Apesar de ser voltada para este tipo de dispositivos, a nova interface também pode ser usada com auxilio de um mouse e teclado. Em torno desta mudança o Internet Explorer 10 foi remodelado para a navegação em tablets. Foi introduzida também uma plataforma com suporte para aplicações em HTML5 e JavaScript o que traz novas oportunidades para desenvolvedores Web. A Microsoft descreve a nova interface como sendo rápida, fluida e dinâmica preservando o poder, flexibilidade e conectividade característica dos sistemas Windows mais recentes. A interface clássica do Windows ainda se encontra presente, podendo o usuário escolher entre esta ou a nova, ou ainda as duas ao mesmo tempo.

Mais detalhes sobre o sistema operacional serão liberados na conferencia para desenvolvedores Build, que será realizada em Setembro. Voce pode também seguir o desenvolvimento pelo twitter: @BuildWindows8.
