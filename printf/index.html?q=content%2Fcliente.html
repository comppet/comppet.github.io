Configurando o Samba como cliente:


De dados


O modo mais fácil de acessar arquivos compartilhados em um servidor Windows após ter instalado o Samba, é pelo Konqueror. Para instalá-lo, basta digitar, no terminal:

$ sudo apt-get install konqueror

Konqueror - Acesso ao servidor Windows


Abra o Konqueror, e na barra de navegação digite smb://ip_servidor.












De impressora

Instalando impressora de rede Windows


Para acessar uma impressora ligada a um servidor Windows, basta abrir a janela de configuração de impressão (Configurações do Sistema -> Impressão) e escolher a opção de adicionar nova impressora. Na janela que abrir, digite o caminho da impressora como na imagem ao lado, ou caso a rede esteja visível, poderá selecionar o caminho pelo "Browse". Agora, basta prosseguir normalmente com a instalação da impressora.
