Software Livre

A Flartoolkit surgiu a partir da biblioteca Artoolkit. Ela é uma biblioteca livre para desenvolvimento de aplicativos em realidade aumentada voltada para aplicações web. Para que o usuário final desfrute das aplicações obtidas a partir desta biblioteca, basta que tenha uma câmera (webcam, por exemplo) e o plugin do Flash instalado em seu navegador. Esta biblioteca funciona independente do sistema operacional que esteja usando, ou seja, não é necessário fazer download e configurá-la na máquina, diferentemente da Artoolkit. Por causa disso, é uma das bibliotecas livres mais utilizada na Internet.

Ela foi desenvolvida em Flash 3D, que, se unida ao Papervision (motor 3D em tempo real que possibilita o desenvolvimento de aplicativos de Realidade Aumentada e Virtual para WEB), pemite o desenvolvimento de aplicações utlizando Action Script, uma linguagem de programação orientada a objetos utilizada principalmente para aplicações na web.
