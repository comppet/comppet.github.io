Uzebox: O video game livre

Você já se imaginou construindo seu próprio videogame, ou então, usando uma plataforma de jogos da qual a comunidade poderia compartilhar projetos de hardware ou de software livremente?

The Uzebox Project, criado pela Belogic, é um projeto denominado Open Source Game Console, ou, Console de Jogos de Código Aberto. Um projeto que vem crescendo desde 2009, com o intúito de integrar pessoas no ramo de desenvolvimento de jogos, facilitar criações de jogos usando a boa e velha linguagem de programação C e relembrar a antiga emoção dos video games retrôs.

O console caseiro visa a simplicidade. Todo o processamento é realizado por um microcontrolador Atmel AVR de 8-bits, que trabalha em conjunto com um kernel de código aberto desenvolvido para este. Uma das características mais chamativas é que o console teve uma implementação dedicada para proporcionar uma boa qualidade de som e gráfico, paleta de 256 cores, com baixo custo.

Além do hardware simples, o video game conta com um requisito interessante: Você poderá reaproveitar seu velho joystick de SNES ou NES. O motivo para isto é facilitar na implementação do projeto, que contaria com um obstáculo e um custo a menos na construção do mesmo, e que já tornaria a jogabilidade bem menos hostil para os jogadores veteranos. O controlador da empresa Nintendo tem, além do objetivo de interagir com o jogo, uma forma de manter todo o projeto com uma característica nostálgica.
