Office Web Apps e Dropbox(){
Dicas

Se você não largaria as ferramentas da Microsoft por nada, não se desespere, ainda há salvação! O Office Web Apps é a aposta da Microsoft para concorrer como Google Docs, mas com um único e grande defeito: não permite a edição simultânea de documentos.

No Office Web Apps é possível criar um documento de Word, Excel, Power Point ou One Note, após criá-lo basta colocá-lo em alguma pasta compartilhada do Microsoft SkyDrive para que todos os membros do grupo possam vê-lo e editá-lo. Simples assim? Não, ainda mais um inconveniente: todos membros do grupo devem ter uma conta com o Office Web Apps ativado, para fazê-lo basta acessar a conta do Windows Live a partir de um computador com o Microsoft Office 2010 instalado.

Há também o cada vez mais popular DropBox, além de ser uma ótima ferramenta de armazenamento em nuvem, o DropBox, assim como o SkyDrive, também permite o compatilhamento de pastas. Portanto, se não for necessário que mais de uma pessoa edite o mesmo arquivo ao mesmo tempo, basta cada componente do grupo ter uma conta no DropBox e uma pasta compartilhada entre todos eles, assim os arquivos dentro dessa pasta serão sempre os mais recentes, eliminando aquela bagunça de versões que se cria na troca de e-mails.
