O que é o Samba?


Logotipo do Samba


O compartilhamento de arquivos multi-paradigma começou quando o Network Basic Input Output System (NetBIOS) foi introduzido como parte de um sistema projetado para criar redes locais. Este protocolo se tornou o Server Message Block (SMB), e atualmente é conhecido como Common Internet File System (CIFS).

Originalmente desenvolvido por Andrew Tridgell em 1992, o Samba é um software livre que pode ser executado em qualquer plataforma diferente do Microsoft Windows, por exemplo, UNIX, Linux, IBM System 390, OpenVMS e outros sistemas operacionais.

É composto por principalmente dois programas: Server Message Block Daemon (SMBD) e Network Message Block Daemon (NMBD). O trabalho deles é implementar quatro serviços básicos:

  • Serviços de arquivo e impressão
  • Autenticação e autorização
  • Resolução de nome
  • Anúncio de serviço (navegação)

Com eles, é possível usar o Linux como servidor de arquivos, servidor de impressão, entre outros, como se a rede utilizasse servidores Windows.

Todo trabalho feito pelo Samba é provido de grande segurança, uma vez que há grande rigor nos controles dos recursos oferecidos. Tanto é que existem empresas que usam o Samba como solução para conflitos existentes entre diferentes versões do Windows.
